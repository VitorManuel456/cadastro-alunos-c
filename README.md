# 🎓 Cadastro de Alunos em C

Projeto desenvolvido para praticar estruturas (`struct`), vetores, funções e menus interativos na linguagem C.

## 📋 Funcionalidades

- ✅ Cadastrar aluno
- ✅ Listar alunos cadastrados
- ✅ Buscar aluno pelo nome
- ✅ Menu interativo
- ✅ Limite de cadastro

## 🛠️ Tecnologias

- Linguagem C

## 📚 Conceitos utilizados

- Struct
- Vetores
- Funções
- Estruturas condicionais (`if`)
- Estrutura de seleção (`switch`)
- Laço de repetição (`do...while`)
- Entrada e saída de dados (`scanf` e `printf`)

## ▶️ Como executar

```bash
gcc src/main.c -o cadastro
./cadastro
```

## 🎯 Objetivo

Praticar estruturas de dados básicas e organização de código utilizando a linguagem C.
